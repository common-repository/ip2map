== Description ==

This plugin add awesome functionalities to track your website visitors GeoIP location by country & city then show you how to display a map of all the 100 most recent visitors to your website or blog. It displays information such as where the visitors came from (city, region and country) and the time of their visits in a table form and graphical map. This plugin is absolutely simple, flexible and free. The geo location information is powered by [IP2Location](http://www.ip2location.com).

There is no registration, sign up, programming or database installation are required in order to use this plugin.


== Installation ==

*Using the WordPress dashboard*

1. Login to your weblog
1. Go to Plugins
1. Select Add New
1. Search for IP2Map
1. Select Install
1. Select Install Now
1. Select Activate Plugin
1. Go to Widgets
1. Drag IP2Map to sidebar

*Manual*

1. Download and unzip the plugin
1. Upload the entire *IP2Map/* directory to the */wp-content/plugins/* directory
1. Activate the plugin through the Plugins menu in WordPress
1. Go to Widgets
1. Drag IP2Map to sidebar
